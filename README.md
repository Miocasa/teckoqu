Next.js "Coming Soon" страница для Miocasa, настроенная под GitHub Pages.

## Скрипты

- `npm run dev` — запуск в режиме разработки
- `npm run build` — сборка проекта (Next.js + статический export в `out/`)
- `npm run deploy` — сборка и деплой в ветку `gh-pages` для GitHub Pages
